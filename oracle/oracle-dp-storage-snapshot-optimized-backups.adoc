= Backups otimizados para Storage Snapshot
:allow-uri-read: 


[role="lead"]
O Oracle Snapshot Optimized Backups era um método de recuperação ponto no tempo que permitia a você realizar uma recuperação usando snapshots sem colocar o banco de dados em modo de hot backup.

Os usuários ainda podem recuperar snapshots sem usar o modo de backup, mas somente a recuperação completa usando RECOVERY AUTOMATIC é suportada. A recuperação incompleta usa o método SNAPSHOT TIME para indicar a hora de criação do snapshot.
